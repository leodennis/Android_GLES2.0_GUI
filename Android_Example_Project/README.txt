The Android_Example_Project is the ideal project to base your next application on.

It provides you with all common GUI elements and you will be able to change them like you want.

The GameManager is the class to look at.
See the initGameElements() to edit the views of the application.


In the issues you can see one issue regarding the GL_Texture:
To prevent this issue just generate the textures with a buffer of 1 pixel on each side.
